# 💫 About Me:
I am a robotics and embedded systems developer specialized in the engineering of autonomous systems, particularly drone technology and robotic simulation. My technical core lies in using ROS and Gazebo to validate complex hardware environments, underpinned by proficiency in Embedded C, C++, and Python. I integrate these systems with spatial computing—including 3D modeling and AR/VR development—while leveraging R and machine learning for advanced modeling. Additionally, I apply full-stack web development skills (HTML/CSS) to create seamless interfaces for robotic applications, ensuring a cohesive bridge between low-level firmware and high-level software.
